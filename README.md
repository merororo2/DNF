# **후기**

## **FrontEnd**
- 클라이언트에서 던파API를 호출하는데 Cors문제가 생겨 어려움이 있었습니다.
- Axios 사용해본 경험이 있어 큰 어려움 없이 수월하게 사용이 가능해 졌습니다.
- 유튜브에 올라와있는 영상을 참조해 간단한 탭메뉴를 어려움없이 구현하게 되었습니다.
- 프로젝트 진행도중 잦은 실수나 오타로 인해 시간을 소비하여 아쉬움이 매우 컸습니다.
